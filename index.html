Thing's you can create with JavaScript

HTML, CSS and JavaScript are the technologies that power the web. You can start making amazing websites once you have these skills in your arsenal. The bare minimum is HTML and CSS, once you know these two languages you can start developing basic websites. JavaScript however opens up the door for more creative and intuitive solutions which take your website to the next level.

For example the game you may have played above was built using JavaScript. Using HTML Canvas and JavaScript you can build your 2d games. Once You've mastered JavaScript you can even build a 3d game using a library called ThreeJS. Don't worry about libraries for now, once you've got a hang of HTML, CSS and JavaScript look into JavaScript frameworks and libraries.

Below are some things you can create using JavaScript.

Websites

Websites can be built using just HTML and CSS. JS allows you to add interactivity. Our navbar changes transparency, this is done using JavaScript.

Games

We built snake using JavaScript. You can make 2d and 3d games using JavaScript. There's a wide range of YouTube tutorials that'll guide through the process.

Mobile Apps

Mobile apps were traditionally built using Java, a different language to JS. In recent years mobile apps have been built using React Native, a JS Framework.

Web Apps

You probably use browser based applications like Google Maps and Deliveroo throughout our day. Many of these apps are built using JavaScript.

Art

HTML Canvas opened up the door for digital art to be created. We used it to create snake but it can also be used to create digital art.

Server Applications

With the creation of Nodejs, JS could be run in the terminal/console. We can now build servers using JS and ExpressJS, a JS framework, to make things easier

Learning web development

Web development is a good place to start before venturing into other JavaScript related fields like Server managment and creating web apps. You will start with HTML, CSS and some basic JavaScript. After you've built a few websites and have some experience under your belt, you can explore all that JavaScript has to offer. Our website aims to help guide you through this learning process by providing you with resources where you can learn in depth knowledge on HTML, CSS and JS. We will also provide you tips on how you can teach yourself.
